## 🎯 Features

### ✅ Current Features
- 📝 **Multiple Choice Questions** - Test your French knowledge
- 📊 **Real-time Scoring** - See your score update instantly
- 🔢 **Progress Tracker** - Know which question you're on
- 🎨 **Modern Design** - Beautiful gradient UI with smooth animations
- 📱 **Fully Responsive** - Works on mobile, tablet, and desktop
- 💡 **Instant Feedback** - Know if you're correct or incorrect
- 🔄 **Restart Option** - Try again anytime

### 🚀 Now live
- ⏱️ **Timer Mode** - Challenge yourself with time limits
- 📚 **Difficulty Levels** - A1, A2, B1, B2 categories
- 💾 **Save Progress** - Continue where you left off
- 🏆 **High Scores** - Track your best performances
- 🔊 **Audio Questions** - Listening comprehension practice
- 📈 **Detailed Analytics** - See your strengths and weaknesses
